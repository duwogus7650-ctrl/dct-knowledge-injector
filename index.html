KNOWLEDGE // INJECTOR

카파시의 도서관, 뒤집어서. "이 모든 책을 다 배울 순 없다 → 그러면 그걸 배울 수 있는 무언가를 만들자." 이 단말기가 그 무언가입니다. 지식을 던지면 대신 소화하고, 인간 뇌가 가장 잘 흡수하는 방식으로 당신 머리에 주입합니다.

주입의 과학(정직하게). 신경망에 지식을 직접 쓰는 뇌 케이블은 아직 없습니다. 그래서 케이블 대신 가장 강력한 실증 학습기제 — 간격반복 + 능동인출 — 을 자동화했습니다. 망각곡선을 거슬러 "잊기 직전"마다 인출시켜 장기기억으로 고정합니다.

엔진: FSRS-5 — Anki에 탑재된, 10억+ 복습 데이터로 학습된 최신 스케줄러. 난이도(D)·안정성(S)·회상확률(R)을 분리 모델링합니다. 좌하단 SELF-TEST가 알고리즘을 6개 레퍼런스값에 대조해 매 로드마다 검증합니다.

흐름: ① 자료 넣기 — PDF·PPTX·DOCX·TXT 끌어놓기, 텍스트 붙여넣기, 또는 자연어로 주제 입력 → ② ✦ Claude로 카드 생성(자료 기반) 또는 🔍 주제만으로 지식 찾기 → ③ INITIATE INJECTION → 복습. 덱은 이 기기에 저장돼 이어집니다. (Claude API 키 필요)

복습 채점(다시/어려움/외웠다/쉬움): 카드 질문을 보고 답을 스스로 떠올린 뒤 정답을 확인하고, 방금 얼마나 잘 떠올렸는지 솔직하게 누릅니다. 앱이 맞다/틀리다 채점하는 게 아니라 본인이 누르는 거예요. 버튼의 "○일"은 그 카드를 다음에 다시 보여줄 시점 — 잘 외운 건 띄엄띄엄, 못 외운 건 자주(잊기 직전마다) 보여줘 최소 노력으로 장기기억에 박습니다.

모델 & 노력: 좌측에서 Haiku 4.5(빠름)·Sonnet 4.6(균형)·Opus 4.8(최고) 중 선택. 노력은 Quick → Standard → Deep → ULTRACODE(멀티패스 + 최대 추론으로 더 깊고 넓게). 모델별 API 규약을 자동 준수합니다.

자연어 토픽 모드: 파일이 없어도 "RAG가 뭔지 처음부터", "스페인어 기초 30개"처럼 물어보면 AI가 그 주제 지식을 찾아 카드로 만듭니다(Claude 키 필요).

100% 클라이언트 사이드(서버 없음). 카드 생성은 Claude API 키로 동작하며, 키는 이 기기에만 저장됩니다. 복습·덱 저장은 오프라인. PDF/PPTX/DOCX는 첫 사용 시 파싱 라이브러리를 1회 내려받습니다.
